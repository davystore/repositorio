# Davy Store Premium 1.1.1

Site simples e moderno para vendas de camisas de time, com botão de compra levando direto para o WhatsApp.

## Arquivos principais
- `index.html` → estrutura do site
- `styles.css` → visual do site
- `assets/logo-davy-store.png` → logo da loja

## Como publicar na Vercel
1. Crie um repositório no GitHub
2. Envie todos os arquivos desta pasta para o repositório
3. Entre na Vercel
4. Clique em **Add New Project**
5. Selecione seu repositório
6. Clique em **Deploy**

Como este projeto é estático, a Vercel normalmente publica sem precisar configurar build.

## Como trocar fotos depois
Hoje os cards estão com a frase `Adicione a foto aqui`.
Depois você pode trocar cada bloco por uma imagem real, por exemplo:

```html
<img src="assets/minha-camisa.png" alt="Camisa Brasil" class="product-image" />
```

Se quiser, depois eu posso te entregar uma segunda versão já preparada com painel mais fácil para trocar fotos e produtos.
